Hex attack on binary

Control: Use the keys Q, W, R, T, Y, U and I to control 8 (0,1) cells at the bottom.

Goal: Make the binary number at the bottom match the decimal number indicated by the "enemies" (green box dropping from the top).

The first 4 cells represent the first digit of the number of enemies, and the next 4 cells represent the last digit.

Binary to Hex:

0000 = 0

0001 = 1

0010 = 2

0011 = 3

0100 = 4

0101 = 5

0110 = 6

0111 = 7

1000 = 8

1001 = 9

1010 = A

1011 = B

1100 = C

1101 = D

1110 = E

1111 = F
